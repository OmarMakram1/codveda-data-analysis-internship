# Codveda Technologies – Data Analysis Internship

This repository contains all tasks I completed during my Data Analysis Internship at Codveda Technologies, following the official task list provided in the program.

---

## 📌 Level 1 (Basic)

### 1. Data Cleaning & Preprocessing
- Handling missing values
- Removing duplicates
- Fixing inconsistent data formats (dates, categories, etc.)
- Tools: Python, pandas
- Notebook: `01_data_cleaning.ipynb`

### 2. Basic Data Visualization
- Bar charts, line charts, scatter plots
- Customizing titles, labels, legends
- Exporting plots as PNG images
- Tools: matplotlib, seaborn
- Notebook: `02_basic_visualization.ipynb`

---

## 📌 Level 2 (Intermediate)

### 3. Linear Regression
- Train/test split
- Building a regression model
- Coefficient interpretation
- Evaluation using R² & MSE
- Tools: scikit-learn, pandas
- Notebook: `03_linear_regression.ipynb`

### 4. Time Series Analysis
- Plotting time series
- Identifying trend and seasonality
- Decomposition
- Moving average smoothing
- Tools: statsmodels, pandas, matplotlib
- Notebook: `04_time_series.ipynb`

---

## 📌 Level 3 (Advanced)

### 5. Classification Models
- Logistic Regression
- Decision Tree
- Random Forest
- Evaluation: Accuracy, Precision, Recall, F1-score
- Hyperparameter tuning using Grid Search
- Tools: scikit-learn, pandas, matplotlib
- Notebook: `05_classification.ipynb`

### 6. Interactive Dashboard (Power BI / Tableau)
- Importing and cleaning the dataset
- Creating interactive charts
- Adding filters and slicers
- Publishing the dashboard
- Tools: Power BI or Tableau
- Folder: `dashboard/`
